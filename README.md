# Reflexcel

Reflexcel is a next-generation software company focused on building innovative products using cutting-edge technologies in AI, Cloud, and Quantum Computing.

## Website Features
- Professional, modern, and responsive design
- Rich content: Home, About, Products, Team, Careers, Contact
- Built with React.js and Material UI
- Ready for deployment on GitHub Pages
- Accessibility and SEO best practices

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Run locally:**
   ```bash
   npm start
   ```
3. **Build for production:**
   ```bash
   npm run build
   ```
4. **Deploy to GitHub Pages:**
   - Update the `homepage` field in `package.json` with your GitHub username.
   - Run:
     ```bash
     npm run deploy
     ```

## Folder Structure
- `src/assets/` — Static assets (logo, images)
- `src/layout/` — Header and Footer components
- `src/pages/` — Main site pages
